Install
=======

Install dependencies: http://pypi.python.org/pypi/flake8

    $ sudo pip install flake8 --upgrade

Install dotfiles:

    $ wget --no-check-certificate https://raw.github.com/limpangel/dotfiles/master/install.sh -O - | /bin/bash
